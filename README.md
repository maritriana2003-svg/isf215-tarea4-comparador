# Sumador/Restador de 4 bits con detección de Overflow

Proyecto individual de Hardware Digital para la asignatura ISF-215.

## Descripción
Este módulo es un circuito combinacional capaz de sumar o restar dos números de 4 bits utilizando el chip 74HC283 y compuertas XOR (74HC86) para la gestión del complemento a 2. Incluye un indicador LED para detección de overflow cuando el resultado supera la capacidad de representación de 4 bits.

## Componentes principales
* Sumador de 4 bits 74HC283
* Compuertas XOR 74HC86
* Indicadores LED para salida y overflow
