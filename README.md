# Tarea 4: Prototipo de Lógica Digital en Simulación
**Asignatura:** ISF-215 Hardware Digital  
**Estudiante:** Adriana Castillo Uribe  

## Descripción del Módulo
Este repositorio contiene el prototipo de un **Comparador de Magnitud de 4 bits** desarrollado en Tinkercad Circuits utilizando el circuito integrado 74HC85. El módulo evalúa de forma combinacional dos números binarios de 4 bits (Número A y Número B), activando una salida exclusiva para indicar de manera visual si A es mayor, menor o igual que B.
